Step 4

Configure Slack Integration

  1. Copy your Railway URL (e.g., https://care-os.up.railway.app)
  2. Go to Slack API
  3. Create your Slack app (see Slack setup guide)
  4. Use your Railway URL for webhooks:
    • Events: https://your-url.railway.app/webhooks/slack/events
    • Interactivity: https://your-url.railway.app/webhooks/slack/interactions
    • Commands: https://your-url.railway.app/webhooks/slack/commands
Step 5

Add Slack Credentials to Railway

  1. In Railway dashboard, click on your project
  2. Go to "Variables" tab
  3. Add these environment variables:

    SLACK_SIGNING_SECRET=your-secret
    PORT=3000
  4. Click "Deploy"
